Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost:9000

Authenticating requests

This API is not authenticated.

    --header "Content-Type: application/json" \

    \"nome_fantasia\": \"lf system\",

URL Parameters

uuid   string   

O Uuid do prédio.

Cadastrando a foto de um predio ou sala

Example request:
curl --request POST \
    "http://localhost:9000/api/v1/predio/1436a969-96b1-4e70-976d-426601fc4d55/foto" \
    --header "Content-Type: multipart/form-data" \
    --form "nome=Foto 1" \
    --form "descricao=Foto 1" \
    --form "tipo=predio" \
    --form "foto=@/tmp/phpC4bM3i" 
const url = new URL(
    "http://localhost:9000/api/v1/predio/1436a969-96b1-4e70-976d-426601fc4d55/foto"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('nome', 'Foto 1');
body.append('descricao', 'Foto 1');
body.append('tipo', 'predio');
body.append('foto', document.querySelector('input[name="foto"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

Example response (200):


{
    "status": true,
    "message": "Registro cadastrado com sucesso",
    "data": {
